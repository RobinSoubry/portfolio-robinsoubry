Project Description

Pioneer is a native iOS application for travel inspiration & discovery. A user can search for any travel destination in the world and swipe left (dislike) or right (like) for each presented highlight. By clicking on a card, the user will get more details about the attraction such as rating, price, address etc.

All liked places are stored in a wishlist. Each location in this list is plotted on a map to provide an overview.

  • Explorers create a detailed profile for their project, link their different social media accounts (Facebook, Instagram, Twitter, etc.) and request sponsorship in an efficient way.
  • Brand Managers receive the requests with key performance indicators on audience size and its fit with the company's policy. He/she can then easily benchmark different projects and grand the fund, gear or other with a few simple clicks.
