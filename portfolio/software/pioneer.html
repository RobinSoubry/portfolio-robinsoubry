Project Description

Pioneer is a native iOS application for travel inspiration & discovery. A user can search for any travel destination in the world and swipe left (dislike) or right (like) for each presented highlight. By clicking on a card, the user will get more details about the attraction such as rating, price, address etc.

All liked places are stored in a wishlist. Each location in this list is plotted on a map to provide an overview.

  • Swiping A key technology is the ability navigate by swiping
  • Data-sources The application sources its destinations from the custom developed Pioneer API and completes cards with Google places if there are not sufficient cards available.
  • Filters The user can filter based on category (restaurants, musea, nature, etc) as well as by radius around a specific destination.

Technology Stack

  • HTML5 & CSS3
