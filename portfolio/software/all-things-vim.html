Project Description

Vim, a productivity tool for programmers has excited many of the world's best programmers. The 'all things vim' shop offers a wide range of merchandise like mugs, t-shirts, cheat sheets etc. The application has been created as an educational game in the 7th week of the devbootcamp curriculum.

  • User authentication & admin roles The application allows different user access levels. The admin profiles can perform all CRUD operations on new articles and manage the inventory level.
  • Product Categories Products can be assigned to distinct categories. A product can belong to many categories
  • Inventory management Each article in the shop has a stock level. A user cannot order more than available, and if the item is out of stock, the 'add to basket' button is disabled automatically
  • Order management Allows the user to check-out. Upon successful completion, the user receives a confirmation e-mail with the details of the order.
